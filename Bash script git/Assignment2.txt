Create a new branch named 'feature':
git checkout -b feature
This command creates a new branch named 'feature' and switches to it.

Make changes to your files in the 'feature' branch. You can edit existing files or create new ones.

Add the changes to the staging area:
git add .
This command adds all changes to the staging area. If you want to add specific files, replace  '.' with the file names.

Commit the changes:
git commit -m "Made changes in the feature branch"
